## 游戏助手

还记得上次折腾游戏相关的事情之后，觉得我游戏方面应该还有作为。

这次呢，我玩新游戏，看见别人发布了一个异环**钓鱼工具**，我觉得好像挺简单。因为肉眼能看出来，是 python 识别信息，然后向游戏发送 “按键” 事件。

第一个想法，是给那个仓库做贡献，但是我看了看架构，研究了下，我搞不懂。它也依赖了一个大的框架。

而我，抱着**学习**的态度，也希望从零到一。

## 快速实现

实现这个工具，最需要的是快速验证想法：
1. 成功捕获画面
2. 成功识别画面
3. 发送按键

过程虽有一些坑，但也算当天实现想法，后续就是慢慢打磨

## 业务实现

第一个重要的业务，钓鱼循环，这套逻辑也遇到蛮多坑。

但是有参考**现有**项目的解决方案 + AI，很快就解决了。

## 业务拓展

参考别人的自动工具，我也做了其它的几个功能。

店长特供这个功能，还有很多优化的，游戏本身难度就不合理，我感觉最后几个关卡，根本不是人能过的。

敲鼓，他这里敲鼓错误的锁定设计非常恶心人。

钢琴这个，反而是找不到乐谱数据，自己也不想弹。

我感觉，只有钓鱼合适。我自己也只挂下钓鱼。

## 总结与新想法

一周，完成了这个项目，推了一些人用，我觉得还可以。不是很想花更多精力去推了，至少我感觉，它做的并不太好，代码在 AI 堆叠上，让我不想再堆了。

### 1. 我想具体的学一下 Python

虽然学过几次，但永远要学下一次，这样写出的东西，我自己才会舒服，“完美感”

### 2. 做新的，绝区零

绝区零，我每天下班回家都是重复的做一套任务下线，我想它自动化去完成，中途我可以吃饭、做事。

这一次，我想设计的更好一点，结合这次的经验。跑通一次，信心就会更稳重，我应该能做的更好。

## 自问

做这些有用吗？虽然，我早已不多想，但，或许有一丝火苗，我都应顺便理清自己行为。

有的：

1. 前端方面，我挖掘太久了

* **UI 方向**，我做很多了。不同角色的人，都反应我做的不错，也有一些受众，应该暂时够了。或许升级，需要下一次契机。
* **后端**方向，工作业务、自己不同情况开发，都有不同经验。并且长期坐下来，我并不太享受后端的东西，或许也需要下次的不同形式，或者有需要我的业务。
* 工作需要的**其它**代码，我习惯看别人的项目，去看人家源代码。去沥青脉络，现在结合 AI，能够更快解析。需要的其它拓展实现，也没有什么困难。年龄心智的提示，很多过去解决不了的问，不想解决的问题，我都一一给出了现在的解法，出好方案。
* 前端的 **3D** 方面，我一次次探寻，我试了很多方案。3D 就算是现在，对于前端，都还算鲜有人优秀的领域。做好真挺难的，有个很现实的问题，就是本身的 3D 体系，很难做出经验的东西，渲染、材质、只能走偏门。
* 前端**本身**，从入行开始，深挖底层，这是那个时期的流行，大家都这么卷。后来，希望做出受欢迎的 Github 仓库，让自己也成为那个 Github 上厉害的人。现在呢，小有成就，但也不厉害，也没那么执着了。

我想说什么~，**啰啰嗦嗦**居然说了这么多。我想说的应该是，我的下一步，没太有下一步的目标。我不知道 3D 这个目标会不会更好，也不知道后端怎样参与的更好，前端卷轮子跟没有必要，我也规划自己做一个 Flutter AI App。

可是，现在，我觉得我做这个游戏工具，使用 Python 视觉做出的 exe 工具，**我蛮喜欢**。

首先它解决了我游戏，重复浪费时间的问题。其次，别人使用，修复bug，让我认知更多方面。一个东西，有人用，好用，这很重要。


2. exe 程序，应该也是前端，我们是**程序开发者**，解决程序需求。AI 时代，在给新时代**重新划定界限**，我不能再继续框定自己的原有的角色范围。

3. 做不一样的事，会遇到不同的人。我希望，我能见**更多**，**形形色色的事**。

生活、工作，往复循环，无聊时间里，我希望喝点不同味道的奶茶。
